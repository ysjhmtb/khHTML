폼 관련 태그

form태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그

form태그 내의 input태그를 통해서 사용자가 입력한 정보를 서버로 전송한다.

사용자의 입력을 받는 부분은 input, 서버로 값을 전송하는 역할은 form태그 이다.

* action : 폼에 입력된 값들을 어디로 전송할지 나타내는 속성(서버 주소)

* method : post/get - 폼에 입력된 값을 서버로 전송 하는 방식

기본 폼 태그 작성

사용자 입력 후 form 태그 내 submit 버튼을 누를 경우 입력 된 값을 서버로 전송한다. (form 태그에 설정된 action 속성의 servlet으로 값을 전달 한다. 전달 된 값은 input의 name 속성을 이용하여 값을 뽑을수 있다.)
